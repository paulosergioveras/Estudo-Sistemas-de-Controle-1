# Estudo-Sistemas-de-Controle-1
Visa criar implementações e funções sobre a disciplina de Sistemas de Controle 1.

## Como rodar - Primeira Ordem

```bash
cd '.\primeira ordem\'  # linux e mac: cd "./primeira ordem/" 

python main.py # linux e mac: python3 main.py
```

## Como rodar - Segunda Ordem

```bash
cd '.\segunda ordem\' # linux e mac: cd "./segunda ordem/"

python -m venv venv # linux e mac: python3 -m venv venv

venv\Scripts\activate # linux e mac: source venv/bin/activate

pip install -r requirements.txt

python main.py # linux e mac: python3 main.py
```
